# energy_agent
EnergyAgent is a Python-based AI agent for detecting hidden energy waste in appliances and usage patterns. Built with Flask and Streamlit, it features Input Understanding, State Tracking, Task Planning, and Output Generation. It delivers real-time streaming output with a modern UI, modular design.
